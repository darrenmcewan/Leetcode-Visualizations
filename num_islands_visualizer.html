Core idea

When the scan finds a 1, that cell is the first visible piece of a new island. The answer increases by one, then DFS sinks the whole connected island.

Why mutation works

Turning land into 0 acts like a visited set. Later scan steps see only water, so the same island is never counted twice.

Neighbor order

The function explores up, then down, then left, then right. The stack panel shows that recursive order.

Complexity

Every cell is inspected by the outer scan and each land cell is sunk once, so the runtime is O(rows × cols).
